YOLO Object Detection API

Upload an image and detect objects using YOLOv8.

Current: ...

How It Works

1

Upload an image using the dropzone below, or send a POST request with a base64-encoded image.

2

The YOLO model processes the image and detects all objects.

3

Results are returned as JSON with class labels, confidence scores, and bounding boxes.

API Reference

MethodEndpointDescription
GET/healthCheck if the server is running
POST/predictDetect objects in a base64-encoded image
GET/guiThis page

cURL Examples

Health check

curl http://localhost:5000/health

Predict from a local file

python3 -c "
import base64, json, sys
with open(sys.argv[1], 'rb') as f:
    b64 = base64.b64encode(f.read()).decode()
print(json.dumps({'image': b64}))
" image.jpg > /tmp/payload.json

curl -X POST http://localhost:5000/predict \
  -H "Content-Type: application/json" \
  -d @/tmp/payload.json
